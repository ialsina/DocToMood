# Changelog

## [0.0.1] - 2026-01-06

### Added
- CLI tool for converting DOCX multiple-choice questions to Moodle XML format
- GUI application for easy file processing
- Support for DOCX and ODT input files
- Multiple correct answer detection methods (highlighting and checkmark)
- Batch processing with glob pattern support
- Output in both Moodle XML and formatted DOCX table
- Configuration file support (`config.yml`)
- Build scripts for standalone executables (Linux, Windows, macOS)
- Command-line options for flexible processing (`--respect-name`, `--no-write`)
- Support for question explanations and notes
- Progress feedback with tqdm

[0.0.1]: https://github.com/ialsina/DocToMoodle/releases/tag/v0.0.1
